## Step 2: Get API Keys (5 minutes)

### OpenAI API Key
1. Visit https://platform.openai.com
2. Sign up / Log in
3. Go to API Keys section
4. Click "Create new secret key"
5. Copy the key

### Twilio Account
1. Visit https://www.twilio.com/try-twilio
2. Sign up (free trial gives $15 credit)
3. Go to Console Dashboard
4. Note down:
   - Account SID
   - Auth Token
5. Go to Messaging â†’ Try it out â†’ Try WhatsApp
6. Note down the WhatsApp sandbox number

```bash
# Test without running server
python test_local.py

# Run the server

python app.py
```

Server runs on http://localhost:5000

## Step 5: Connect WhatsApp (5 minutes)

### For Local Testing:

1. **Install ngrok**: https://ngrok.com/download
2. **Run ngrok**:
   ```bash
   ngrok http 5000
   ```
3. **Copy the HTTPS URL** (e.g., `https://abc123.ngrok.io`)
4. **Configure Twilio Webhook**:
   - Go to Twilio Console
   - Messaging â†’ Try it out â†’ Try WhatsApp
   - Sandbox settings
   - Set "WHEN A MESSAGE COMES IN" to: `https://abc123.ngrok.io/webhook`
   - Save

5. **Join the sandbox**:
   - Send the join code from your WhatsApp to the Twilio number
   - Example: "join happy-dog"

6. **Test it!**
   - Send: "I had 2 rotis and dal"
   - You should get a response with calories!
